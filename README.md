# Program Rata Rata Nilai 
 Program Rata Rata Nilai digunakan untuk menghitung rata-rata nilai siswa berdasarkan jumlah siswa dan nilai masing-masing yang dimasukkan oleh pengguna.
CARA KERJA:
- Program menggunakan Scanner untuk membaca input dari keyboard.
- Pengguna memasukkan jumlah siswa.
- Program menggunakan perulangan for untuk membaca nilai setiap siswa.
- Nilai-nilai dijumlahkan ke dalam variabel totalNilai.
- Rata-rata dihitung dengan rumus:
\mathrm{Rata-rata}=\frac{\mathrm{Total\  Nilai}}{\mathrm{Jumlah\  Siswa}}- Hasil ditampilkan dengan format dua angka di belakang koma.
